<Steps>
  <Step title="Access the Console">
    Log in to your **[XA Hosting Game Panel](https://panel.xahosting.co/)** and select your server.
  </Step>
  <Step title="View Server Logs">
    In the console, you can see live server logs including player activity, chat, and system messages. Use this to monitor your server in real-time.
  </Step>
  <Step title="Send Commands">
    You can type Minecraft server commands directly into the console input to manage your server instantly. For example, `/op {username}` or `/gamemode creative {username}`.
  </Step>
</Steps>
