<Steps>
    <Step title="Open server.properties">
    - Go to your **File** in the game panel.  
    - Locate the `server.properties` file for your server.
    </Step>
    <Step title="Edit the MOTD">
    - Find the `motd` field inside the file.  
    - Replace it with your desired message (Mostly found in Line 36). Example:
    <CodeGroup>
    ```javascript server.proparties
    motd=I like bananas...
    ```
    </CodeGroup>
    </Step>
    <Step title="Save and Restart">
    - Save the `server.properties` file.  
    - Restart your server to apply the new MOTD.
    </Step>
</Steps>

<Check>
Players will now see your customized MOTD when joining the server.
</Check>
