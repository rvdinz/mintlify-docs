---
title: 'Installing Plugins'
description: 'Easily install and manage plugins on your CraftLands Hosting server.'
icon: 'puzzle-piece'
---

<Steps>
    <Step title="Open the Plugin Installer">
    - Log in to your **Server Dashboard**.  
    - Select **Plugin Installer** from the left-hand menu.
    </Step>
    
    <Step title="Search for Plugins">
    - Use the search bar to browse plugins from trusted repositories like **SpigotMC** or **Bukkit**.  
    - Check the plugin’s description to ensure it’s compatible with your server version.
    </Step>
    
    <Step title="Install the Plugin">
    - Click **Install** next to the plugin you want.  
    - The system will automatically download and configure it for your server.
    </Step>
    
    <Step title="Restart Your Server">
    - Once installed, restart your server to activate the plugin.  
    - You’re ready to use it in-game!
    </Step>
</Steps>

<Check>Your plugin is now live, customize your server and make it unique!</Check>
