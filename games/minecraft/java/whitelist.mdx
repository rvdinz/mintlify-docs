---
title: 'Whitelist Management'
description: 'Learn how to enable and manage a whitelist on your XA Hosting Minecraft server to control who can join.'
icon: 'list-ul'
---

<Steps>
    <Step title="Turn on the Whitelist">
    - Open your server **Console** in the XA Hosting panel.  
    - Run the command:  
      ```bash
      /whitelist on
      ```
    </Step>
    
    <Step title="Add Players to the Whitelist">
    - To give a player access, type:  
      ```bash
      /whitelist add {player_name}
      ```
    - Replace `{player_name}` with their exact Minecraft username.
    </Step>
    
    <Step title="(Optional) Turn Off the Whitelist">
    - If you no longer need it, disable the whitelist with:  
      ```bash
      /whitelist off
      ```
    </Step>
</Steps>
