## Minecraft Server

<Steps>
  <Step title="Choose Your Minecraft Version">
    Go to the **Version Changer** in our Game Panel. Select the Server Software & Minecraft Version you want to run.
  </Step>
  <Step title="Launch Your Server">
    Click **Start** to boot up your server.
  </Step>
  <Step title="Monitor with the Console">
    Go to the **Console** tab to view server logs, for any errors.  
  </Step>
</Steps>

<Check>You've setup your minecraft server for the first time. Congrats!</Check>
